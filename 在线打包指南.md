# 📱 在线打包APK指南

由于本地编译需要安装Android SDK（约2GB），推荐使用以下在线方案：

---

## 方案一：GitHub Actions（推荐，完全免费）

### 步骤：

1. **注册GitHub账号**
   - 访问 https://github.com 注册

2. **创建新仓库**
   - 点击右上角 "+" → "New repository"
   - 仓库名: `CardCounter`
   - 设为Public公开仓库
   - 点击 "Create repository"

3. **上传项目文件**
   - 下载并安装 [Git](https://git-scm.com/downloads)
   - 在项目目录右键 → "Git Bash Here"
   ```bash
   cd D:\Software\斗地主记牌器\AndroidAPP
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/你的用户名/CardCounter.git
   git push -u origin main
   ```

4. **触发构建**
   - 进入GitHub仓库页面
   - 点击 "Actions" 标签
   - 选择 "Build APK" workflow
   - 点击 "Run workflow" 按钮

5. **下载APK**
   - 等待约3-5分钟
   - 构建完成后在页面底部找到 "Artifacts"
   - 下载 `CardCounter-APK.zip`
   - 解压得到 `CardCounter_v1.0.apk`

---

## 方案二：Replit（无需安装）

1. **访问Replit**
   - https://replit.com

2. **创建新项目**
   - 选择 "Android" 模板
   - 或导入现有项目

3. **上传代码**
   - 将AndroidAPP文件夹内容上传

4. **运行构建**
   ```bash
   ./gradlew assembleRelease
   ```

5. **下载APK**

---

## 方案三：AppyPie / AppsGeyser（网站打包）

### AppyPie
- 网站: https://www.appypie.com/
- 操作: 上传文件 → 选择模板 → 生成APK
- 费用: 免费（有水印） / 付费（无水印）

### AppsGeyser
- 网站: https://www.appsgeyser.com/
- 操作: 创建应用 → 上传文件 → 生成APK
- 费用: 完全免费
